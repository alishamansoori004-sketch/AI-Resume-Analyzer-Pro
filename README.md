# AI Resume Analyzer Pro

AI Resume Analyzer Pro is a Flask-based web application that analyzes resumes and provides:

* ATS Score
* Skill Detection
* Recommended Skills
* Career Recommendations
* Improvement Suggestions

## Features

* Upload PDF resumes
* Extract resume text
* Analyze skills automatically
* Generate ATS score
* Suggest improvements
* Responsive modern UI

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* PDFPlumber
